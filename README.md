# PID-Tuning-Ziegler-Nichols-GA-GSA
This repository contains MATLAB implementations of PID controller tuning and optimization using the Ziegler–Nichols method, Genetic Algorithm (GA), and Gravitational Search Algorithm (GSA). The accompanying video demonstrates how to use these codes.

The Gravitational Search Algorithm (GSA) implementation is based on the following paper:
E. Rashedi, H. Nezamabadi-pour, and S. Saryazdi, "GSA: A Gravitational Search Algorithm," Information Sciences, vol. 179, no. 13, pp. 2232–2248, 2009.

This repository contains an unpublished Persian-language manuscript authored by Vahid Mohammadnezhad under the supervision of Dr. Maryam Zekri in 2011. The document is made available for educational and research purposes.
